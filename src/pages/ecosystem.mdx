# Ecosystem

<Details icon='contribute' label='Help us make this page better!'>
  Hi there, so we're looking for libraries and tools that can do the following:
  
  - useful things for 3d rendering / graphics / video-gaming in browsers
  - edge cases / difficult things to do in frontend apps
  - niche libraries that just REALLY fill out their use-cases
</Details>

React boasts a vast ecosystem of open-source libraries, plugins, and features that can help your team rapidly develop UIs and applications.

Generally, to make our list below, the library in question needs to have support and active development/maintenance. The rules are not strict, something like 100K+ weekly downloads on npm and a release within the last 6-12 months.

## Build Tooling, Linting, and Formatting

If your team wants a modern, high-performance toolchain, these are worth evaluating together:

- <Resource url='https://www.npmjs.com/package/vite'>vite</Resource> - fast dev server and build tool for React projects
- <Resource url='https://www.npmjs.com/package/rolldown'>rolldown</Resource> - Rust-powered bundler option in the modern Vite ecosystem direction
- <Resource url='https://www.npmjs.com/package/biome'>biome</Resource> - all-in-one formatter + linter (and more) with strong performance
- <Resource url='https://www.npmjs.com/package/oxlint'>oxlint</Resource> - very fast linting focused on large codebases
- <Resource url='https://www.npmjs.com/package/oxfmt'>oxfmt</Resource> - fast formatting companion from the same ecosystem

### Suggested adoption path

1. Start with <Resource url='https://www.npmjs.com/package/vite'>vite</Resource> for project scaffolding and dev/build performance.
2. Use <Resource url='https://www.npmjs.com/package/zod'>zod</Resource> for runtime-safe schemas in forms and API boundaries.
3. Choose one formatting/linting stack:
   - unified approach: <Resource url='https://www.npmjs.com/package/biome'>biome</Resource>
   - split approach: <Resource url='https://www.npmjs.com/package/oxlint'>oxlint</Resource> + <Resource url='https://www.npmjs.com/package/oxfmt'>oxfmt</Resource>
4. Evaluate <Resource url='https://www.npmjs.com/package/rolldown'>rolldown</Resource> as your bundling strategy matures.
